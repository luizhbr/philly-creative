# assets/images

Pasta organizada com as imagens do material da Philly Creative.

## Estrutura

```
assets/images/
├── hero/              # Imagem principal do topo (Hero)
├── sections/          # Prints das seções do site (Planos, Serviços, Sobre, etc.)
├── portfolio/         # Exemplos de personalização e artes criadas
├── clients/           # Logos e depoimentos de clientes
└── others/            # Outras imagens e screenshots
```

## Como usar

- `hero/inicio.png` → Imagem principal da página inicial
- `sections/planos.png` → Print da seção de pacotes
- `sections/nossos-servicos.png` → Print da seção de serviços
- `clients/nossos-clientes*.png` → Logos e depoimentos para prova social
- `portfolio/*` → Exemplos de trabalhos personalizados

## Observação

As imagens são screenshots do material impresso original da Philly Creative. Elas podem ser usadas para:
- Galeria de "Nosso Trabalho"
- Referência visual
- Futura seção de portfólio

Mantenha os nomes limpos e organizados nas subpastas.
